# 3arrasli.tn

Project structure:

- frontend: React landing page (Vite)
- backend: Flask API

## Run frontend (landing page)

1. cd frontend
2. npm install
3. npm run dev
4. Open http://localhost:5173

## Run backend (Flask)

1. cd backend
2. python -m venv .venv
3. .venv\Scripts\activate
4. pip install -r requirements.txt
5. python app.py
6. API: http://localhost:5000/api/health
